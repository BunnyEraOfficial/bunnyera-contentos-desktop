# BunnyEra ContentOS — Desktop Edition (v0.1)

A multi‑platform, multi‑account content operating system designed for
visual creators, automation workflows, and system‑level brand
operations. This is the desktop version of the BunnyEra ecosystem.

This repository contains the Electron‑based application that manages:
- Local content library
- Multi‑platform publishing
- Template generation
- Queue management
- Logs and system signals
- Integration with BunnyEra Core Backend (Alibaba Cloud)

---

## ✅ Project Goals

1. Provide a unified desktop console for all BunnyEra content operations.
2. Support multi‑platform publishing (Pinterest, IG, X, Telegram).
3. Integrate with BunnyEra Core Backend for automation and logs.
4. Deliver a cyber‑inspired UI consistent with BunnyEra’s brand.
5. Serve as the foundation for future Matrix / Console modules.

---

## ✅ MVP Scope (v0.1)

### ✅ Local Library Viewer
- Read folders from BunnyEra_ContentOS local directory.
- Display categories: Wallpapers, SystemUI, BrandVisuals, etc.
- Preview images inside the app.

### ✅ Template Generator
- Auto‑fill title / description / tags based on category.
- Copy to clipboard with one click.

### ✅ Manual Publishing Helper
- Select platform (Pinterest / IG / X).
- Copy metadata for manual upload.
- Mark items as “Published”.

### ✅ Logs
- Local JSON log of all publishing actions.
- Console‑style UI (BunnyEra aesthetic).
